nmap - Linux

System üzerinde açık olan portları ve kullanan programlar hakkına bilgi alır. nmap.txt

Args Description
-p Sadece özel portları tara
-sV Portları tarar servis ve versiyon bilgisi verir.
-O OS detection, OS versiyon bilgisi verir.
-T Hızlı veya yavaş taratma (0 - 5 (Hızlı - Yavaş))
-A OS detection, OS versiyonu, Script taraması, Traceroute taraması
-v Detaylı bilgi verir.
-vv Çok Detaylı bilgi verir.
-Pn Tüma ana bilgisyarı çevrim içi olarak kabul et
--script Script taraması yapar.
--help Yardım döküman listesi
-V Versiyon numarası

Examples Commands

Usage Description
nmap [OPTION] Varargs Syntax
nmap --help Yardım listesini getir.
nmap -V Versiyon numarsını getir.
nmap -p- 192.168.1.43 Tüm portları tara
