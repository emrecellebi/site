Args Description
-v, --verbose Detaylı bilgi verir.
--help Yardım döküman listesi

Examples Commands

Usage Description
chmod [OPTION] MODE [FILE] Varargs Syntax
chmod --help Yardım listesini getir.
chmod --version Versiyon numarsını getir.
chmod 644 getflag.py -rw-r--r-- Owner okuma, yazma izni - Diğerleri okuma izni
chmod 666 getflag.py -rw-rw-rw- Herkes için okuma, yazma izni
chmod 700 getflag.py -rwx------ Owner okuma, yazma, çalıştırma izni
chmod 777 getflag.py -rwxrwxrwx Herkes için okuma, yazma, çalıştırma izni
chmod +x getflag.py -rwxrwxrwx Herkes için okuma, yazma, çalıştırma izni
