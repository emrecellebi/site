mkdir - Linux Temel Komutu

Terminal üzerinde klasör yaratmak için kullanılır. mkdir.txt

Args Description
-p, --parents Alt dizinlari oluşturur.
-v, --verbose Oluşturulan her dizin için bir mesaj yazar.
--help Yardım döküman listesi
--version Versiyon numarası

Examples Commands

Usage Description
mkdir --help Yardım listesini getir.
mkdir --version Versiyon numarsını getir.
mkdir -v test Test directory oluşturulduğunda bilgi verir.
mkdir -p test/text2 Tüm alt directory oluştrur.
