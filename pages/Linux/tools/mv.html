mv - Linux Temel Komutu

Terminal üzerinden dosya veya dizin taşıma işlemi yapar. mv.txt

Args Description
-v, --verbose Taşıma yaparken bilgi verir.
--help Yardım döküman listesi
--version Versiyon numarası

Examples Commands

Usage Description
mv [OPTION] [SOURCE] [DEST] Varargs Syntax
mv --help Yardım listesini getir.
mv --version Versiyon numarsını getir.
mv -v 01.txt test/01.txt Taşıma işlemini yaparken detaylı bilgi ver.
