rm - Linux Temel Komutu

Terminal üzerinden dosya veya dizin sileme işlemi yapar. rm.txt

Args Description
-f, --forge Var olmayan dosyaları ve argümanları görmezden gel, asla sorma
-R, -r, --recursive Recursive olarak silme işlemi yapar.
-v, --verbose Silme yaparken bilgi verir.
--help Yardım döküman listesi
--version Versiyon numarası

Examples Commands

Usage Description
rm [OPTION] [FILE] Varargs Syntax
rm --help Yardım listesini getir.
rm --version Versiyon numarsını getir.
rm -v 01.txt Silme işlemini yaparken detaylı bilgi ver.
rm -r test Tüm directory recursive olarak sil
rm -f test Hiçbir şey sormadan zorla sil
