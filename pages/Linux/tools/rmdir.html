rmdir - Linux Temel Komutu

Terminal üzerinde dizin silmek için kullanılır. rmdir.txt

Args Description
-p, --parents Alt dizinlari siler.
-v, --verbose Silinen her dizin için bilgi verir.
--help Yardım döküman listesi
--version Versiyon numarası

Examples Commands

Usage Description
rmdir --help Yardım listesini getir.
rmdir --version Versiyon numarsını getir.
rmdir -v test Test directory silindiğinde bilgi verir.
rmdir -p test/text2 Tüm alt directory sil.
