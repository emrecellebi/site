apt-get - Linux Temel Komutu

Paket yükleyicisi apt-get.txt

Args Description
install Paketi yükle
reinstall Paketi yeniden yükle
remove Paketi sil
autoremove Paketi otomatik sil
update Listedeki paketleri güncelle
upgrade Systemdeki paketleri yükle/güncelle
clean Archive dosylarını temizle
autoclean Archive dosylarını otomatik temizle
--fix-mission Hata alınsa bile paketi yükle
--help Yardım döküman listesi
--version Versiyon numarası

Examples Commands

Usage Description
apt [OPTION] [command] Varargs Syntax
apt --help Yardım listesini getir.
apt --version Versiyon numarsını getir.
